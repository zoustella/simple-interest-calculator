# simple-interest-calculator
Simple Interest CalculatorA simple tool to calculate the interest earned over a period of time.

Formula:The calculator uses the standard formula:
$$A = P(1 + rt)$$

Where:  
$P$ = Principal amount  
$r$ = Annual interest rate  
$t$ = Time in years  

Usage:
1. Clone the repository.
2. Run the script (e.g., python calculator.py).
3. Enter the required values.
